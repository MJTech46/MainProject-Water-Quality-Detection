# ✅ Step 2 — Normalize Each Parameter (0 to 1 Scale)

We convert each into a **quality factor between 0 and 1**.

---

### 🔹 Temperature Factor

[
T_f = 1 - \frac{|Temp - 25|}{\Delta T}
]

Where:

* ΔT = 10°C (tolerance range)

Clamp between 0 and 1.

---

### 🔹 TDS Factor

[
TDS_f = 1 - \frac{TDS}{TDS_{max}}
]

Let:

* TDSmax = 1000 ppm

Clamp between 0 and 1.

---

### 🔹 Turbidity Factor

If already % clarity:

[
Turb_f = \frac{Turbidity%}{100}
]

If using NTU:

[
Turb_f = 1 - \frac{NTU}{NTU_{max}}
]

# ✅ Final Refactor Index (New RI Equation)

We combine using weighted average.

Turbidity and TDS impact safety more than temperature.

Let’s assign weights:

* Turbidity → 0.4
* TDS → 0.4
* Temperature → 0.2

## 🎯 New Equation:

[
RI = 100 \times \left( 0.4Turb_f + 0.4TDS_f + 0.2T_f \right)

# ✅ Final Practical Formula

[
RI = 100 \times \left(
0.4\frac{Turb%}{100}
+
0.4\left(1 - \frac{TDS}{1000}\right)
+
0.2\left(1 - \frac{|Temp - 25|}{10}\right)
\right)
]

Clamp RI between 0 and 100.

# ✅ JavaScript Implementation

Replace your compute function with:

```javascript
function computeRI(temp, tds, turbPercent) {

  const TDS_MAX = 1000;
  const TEMP_IDEAL = 25;
  const TEMP_RANGE = 10;

  let turbFactor = turbPercent / 100;
  let tdsFactor = 1 - (tds / TDS_MAX);
  let tempFactor = 1 - (Math.abs(temp - TEMP_IDEAL) / TEMP_RANGE);

  // Clamp between 0 and 1
  turbFactor = Math.max(0, Math.min(1, turbFactor));
  tdsFactor = Math.max(0, Math.min(1, tdsFactor));
  tempFactor = Math.max(0, Math.min(1, tempFactor));

  let ri = 100 * (
    0.4 * turbFactor +
    0.4 * tdsFactor +
    0.2 * tempFactor
  );

  return Math.round(Math.max(0, Math.min(100, ri)) * 10) / 10;
}
```
